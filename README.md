
# AutoApply AI - Automated Job Application Email Generator

This is a Streamlit-based web app that helps you generate personalized job application emails automatically.  
Simply enter the job details, your skills, contact info, and get a professional email draft tailored for the role.

## Features

- Input job title, job description, company name, hiring manager name, your name, phone, email, and skills.
- Validates mandatory fields: Job Title, Job Description, Your Name, Email, Phone.
- Validates email format and Indian phone number format (10 digits starting with 6-9).
- Generates a customized, professional job application email.
- Beautiful baby pink themed UI for a pleasant experience.
